# Railway Deployment Template

Deze file dient als referentie voor het configureren van Railway environment variabelen.

## Required Environment Variables

Zet deze variabelen in het Railway Dashboard onder "Variables":

### Node.js Runtime
```
NODE_ENV=production
PORT=8080
```

### Supabase Configuration
```
SUPABASE_URL=https://bktbeefdmrpxhsyyalvc.supabase.co
SUPABASE_SERVICE_KEY=<jouw-service-role-key>
SUPABASE_ANON_KEY=<jouw-anon-key>
```

### Zwift Authentication (optioneel - voor sync scripts)
```
ZWIFT_USERNAME=<jouw-zwift-email>
ZWIFT_PASSWORD=<jouw-zwift-wachtwoord>
```

### ZwiftRacing API (optioneel - voor race data)
```
ZWIFTRACING_API_KEY=<jouw-api-key>
```

### Frontend Environment (Vite build)
```
VITE_SUPABASE_URL=https://bktbeefdmrpxhsyyalvc.supabase.co
VITE_SUPABASE_ANON_KEY=<jouw-anon-key>
```

## Quick Copy Commands

Voor Railway CLI:
```bash
railway variables --set "NODE_ENV=production"
railway variables --set "PORT=8080"
railway variables --set "SUPABASE_URL=https://bktbeefdmrpxhsyyalvc.supabase.co"
railway variables --set "SUPABASE_SERVICE_KEY=<your-key>"
railway variables --set "SUPABASE_ANON_KEY=<your-key>"
railway variables --set "VITE_SUPABASE_URL=https://bktbeefdmrpxhsyyalvc.supabase.co"
railway variables --set "VITE_SUPABASE_ANON_KEY=<your-key>"
```

## Belangrijke Notes

1. **Secrets in GitHub**: Voeg `RAILWAY_TOKEN` en `RAILWAY_APP_URL` toe aan GitHub Secrets
2. **Supabase Keys**: Vind deze in Supabase Dashboard → Settings → API
3. **Railway Token**: Genereer in Railway Dashboard → Account Settings → Tokens
